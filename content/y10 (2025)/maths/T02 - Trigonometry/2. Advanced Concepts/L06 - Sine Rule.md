# Sine Rule

> [!info] Sine Rule
> $$\frac{a}{sin(A)} = \frac{b}{sin(B)} = \frac{c}{sin(C)}$$
> To use the sine rule, we need:
> - one angle-side pair, **and**
> - either a side length OR an angle

> [!tip] Alternate Sine Rule
> $$\frac{sin(A)}{a} = \frac{sin(B)}{b} = \frac{sin(C)}{c}$$
> - This is used to find an angle instead of a side.
> - This version is an easier way of finding an angle without having to rearrange the formula yourself.
